Source

These recognition results come from the Stock Symbol Document Collection.

Fields

Each recognition result has the following fields:

Property Name Type Description
symbol String The stock symbol
companyName String The name of the company, in English
exchange String The stock exchange the symbol is listed on
marketCap Number The market capitalization, on July 16th, 2014, in dollars
ipoYear Number The year of the company's IPO
industry String The industry the company belongs to
sector String The sector the company belongs to

Example

{
  "com.solveforall.recognition.finance.stocks.StockSymbol": [
    {
      "symbol":  "YHOO",
      "companyName": "Yahoo! Inc.",
      "exchange": "NASDAQ",
      "marketCap": 34020046847.86,
      "ipoYear": 1996,
      "industry": "EDP Services",
      "sector": "Technology"
    }
  ]
}
